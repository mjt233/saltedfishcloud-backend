# download

## 简介

离线下载模块已从主程序分离为独立的插件（`sfc-ext-download`）。该插件提供基于 HTTP/HTTPS 协议的离线下载功能，支持将远程文件下载到网盘指定目录。

## 主要功能

- 支持通过 URL 创建下载任务，文件自动保存到网盘
- 支持 HTTP 代理下载（使用系统中配置的代理节点）
- 支持查看下载进度（实时速度、已下载量、总大小）
- 支持中断正在进行的下载任务
- 支持按用户查询下载任务列表，并可按状态筛选（ALL / DOWNLOADING / FINISH / FAILED）
- 自动处理文件名冲突（重名文件添加 `.1`、`.2` 等后缀）

## 配置

插件的代理配置复用了系统已有的代理节点配置，可在管理员后台或个人代理节点管理中进行配置，也可以在选择节点时快捷配置。
