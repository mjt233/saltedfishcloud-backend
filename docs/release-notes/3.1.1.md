# v3.1.1 版本更新记录

## 变更概况

- 发布版本: `v3.1.1`
- 变更日期范围：`2026-05-07` ~ `2026-05-13`

## 主要更新内容

---

### 功能更新与优化

- 系统支持脱离 MySQL 与 Redis 运行，新增基于 SQLite 的数据库支持和基于本地内存的缓存与消息队列支持，满足轻量化部署和本地开发需求。

### Bug修复

- 修复 `WebDAV` 无法获取当前会话登录用户对象的问题，解决Windows资源管理器挂载时无法创建目录、重命名文件的问题。
- 修复文件收集表结构与多数据库兼容相关问题，提升 SQLite 场景稳定性。
- 修复本地消息队列、缓存及若干并发场景下的稳定性问题，避免任务消费与队列处理异常。
- 修复部分文档与配置说明不一致的问题，减少升级和使用过程中的歧义。

### 代码架构调整/重构

- 重构 RPC 相关实现，拆分 `RPCManager` 职责，优化 `RPCInvoker` / `RPCRegistry` 的边界。
- 重构缓存体系，将核心业务逐步迁移到统一的 `CacheService`、`LockFactory` 与相关抽象接口。
- 更新版本号并同步相关构建、部署与发布流程的支持逻辑。

## 其他说明

- 本次发布继续以 `Java 25` 为运行基线，部署环境需确认 JDK 版本兼容。
- 当前版本涉及缓存、本地消息队列、SQLite 与归档能力的较多重构，建议在预发环境重点验证数据库、文件系统、异步任务与插件链路。
