## 变更概况

- 发布版本: 3.1.2
- 发布日期: 2026-07-05
- 变更日期范围：2026-05-13 ~ 2026-07-05

## 主要更新内容

---

### 功能更新与优化

- **MCP 模块**: 全新 MCP 插件，提供 OAuth 认证、文件搜索、上传下载等工具，支持独立文件上传下载 Controller
- **OAuth/OIDC**: 实现标准 OIDC 服务端接入，支持设备授权模式、永久 ApiTicket、第三方应用授权码流程
- **数据管理插件 (data-manager)**: 全新插件，支持失效数据检测（UNIQUE/RAW模式）、文件类型识别（Tika/FFMpeg）、Groovy 脚本筛选、数据认领/修复/清理等完整功能
- **评论系统**: 实现评论回复功能，支持内容安全过滤、发送频率控制、IP 显示配置、回复时 @ 用户
- **PXE 启动插件 (pxe-boot)**: 全新插件，支持 iPXE 脚本、UEFI 固件、UDF ISO 解析、WIMBOOT 引导 Windows 启动、Proxy DHCP 增强兼容性
- **文件系统重构**: 存储接口语义重构，引入 StorageFactory/StorageRegistry 机制，统一存储工厂管理，移除 TempStoreService 全面迁移到 AttachStorage
- **跨网盘文件操作**: 开放平台存储接口支持跨网盘复制文件，文件移动接口支持跨用户网盘移动
- **文件临时下载链接**: 新增文件临时下载链接接口，使用 JWT 生成授权码
- **桌面组件**: 支持配置作用范围
- **TypeUtils**: 补充日期格式化相关方法
- **配置节点**: inputType 支持 textarea 类型
- **数据库连接重试**: 启动时允许 3 分钟内重试连接
- **插件机制**: 支持加载插件下的 application.yml 合并到主程序配置

### Bug修复

- 修复 OAuth/OIDC 授权流程多个问题（token 生成、授权端点、登录跳转、scope 信息等）
- 修复 MCP API Key 无权限调用存储读写接口问题及哈希算法改为 SHA-256
- 修复共享链接在私人网盘下他人无法访问的问题
- 修复 DBUtils.batchInsert 枚举序列化问题
- 修复插件第三方库干扰系统启动触发 NoClassDefFoundError 的问题
- 修复全局异常处理未处理 http status 状态码的问题
- 修复文件极速传上传报错问题
- 修复文件路径列出不报错、路径校验逻辑等问题

### 代码架构调整/重构

- **文件系统重构**: DirectRawStoreHandler 重命名为 Storage，StoreReader/Writer 合并，DiskFileSystemFactory 重命名为 StorageFactory，新增 StorageRegistry 统一管理
- **评论重构**: JPQL 改用多行字符串，listByTopicId 方法参数调整，桌面组件名称调整
- **OAuth/OIDC 重构**: 移除原有 ApiTicket 机制改用标准 OAuth 实现，OidcJwkService 移除改用 JWK 自动生成持久化，移除 OidcAuthorizationService
- **Config 重构**: 移除 SysConfigName 常量类，配置项 key 通过配置实体类 Lambda 化
- **安全选项配置**: 提取为统一配置类
- **移除废弃代码**: SambaDiskFileSystem、DiskFileSystem 多余方法、文件记录同步功能等

## 其他说明

- Maven 版本号管理规范化，使用 revision 统一主版本号管理
- 数据库新增 OAuth2 授权相关表（Access Token、OIDC 元数据持久化）
- 新增数据管理插件和 PXE 启动插件，初始化其数据库表结构
