**接口路径**: `POST /api/openApi/diskFile/upload/v1`

**权限要求**: `storage_write`

**功能说明**: 上传文件到授权用户网盘的指定目录。

**请求格式**: `multipart/form-data`

**请求参数**:

| 参数名 | 类型   | 必填 | 说明               |
|--------|--------|------|--------------------|
| `uid`  | number | 是   | 目标用户ID，`0` 表示公共网盘 |
| `path` | string | 是   | 目标目录路径       |
| `file` | file   | 是   | 要上传的文件内容   |

**成功响应示例**:
```json
{
    "code": 200,
    "businessCode": 200,
    "msg": "OK",
    "data": 1
}
```

**说明**: `data` 为操作结果码：`1` 表示新文件，`0` 表示已覆盖同名文件，`2` 表示文件内容无变化。
